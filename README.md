# Generador Seguro de Contraseñas

## Descripción del proyecto

Este proyecto es un Generador Seguro de Contraseñas desarrollado en Python.  
Su objetivo es crear contraseñas seguras de forma automática, permitiendo al usuario seleccionar la longitud y los tipos de caracteres que desea utilizar.

El sistema ayuda a mejorar la seguridad digital evitando el uso de contraseñas débiles o repetidas.

## Objetivo

Desarrollar un sistema que genere contraseñas seguras mediante el uso de estructuras de programación como:

- Funciones
- Condicionales
- Bucles
- Validaciones de entrada

## Funcionalidades

✔ Ingresar longitud de la contraseña  
✔ Seleccionar tipos de caracteres:
- Mayúsculas
- Minúsculas
- Números
- Símbolos especiales  

✔ Generación automática de contraseña  
✔ Validación de entrada del usuario  
✔ Opción de generar nuevas contraseñas  
✔ Copiar contraseña al portapapeles  


## Tecnologías utilizadas

- Python 3
- Librería `random`
- Librería `string`
- Librería `pyperclip`
